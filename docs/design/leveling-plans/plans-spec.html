Row anatomy
  • Leading glyph encodes status at a glance: ▸ in-progress · ⚠ stale · ✓ done · ○ never walked.
  • Status pill is the resting display; on hover/selection it swaps for the action triplet so the row never grows tall.
  • Hypothetical plans render the character cell as the italic literal "hypothetical" — they are not filtered out, they are walkable.
  • Selected row gets a 2 px teal left edge; that is the only place the accent appears in this view besides the in-progress progress bar.
